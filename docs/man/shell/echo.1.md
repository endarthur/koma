# NAME

echo - display text

## SYNOPSIS

`echo [text...]`

## DESCRIPTION

Display arguments as text to the terminal. Joins all arguments with spaces.

## EXAMPLES

Print a message:
```
echo Hello, World!
```

Print multiple words:
```
echo The quick brown fox
```

Print with quotes:
```
echo "This is a quoted string"
```

## SEE ALSO

write(1), cat(1)
